# SaaS Landing Page Template

A complete, production-ready landing page template designed for SaaS products and startups.

## Included Sections

1. **Hero — Split Image**
   - Bold headline with supporting copy
   - Primary and secondary CTAs
   - Large product screenshot

2. **Feature Grid**
   - 3-column responsive layout
   - Icon + heading + description per feature
   - Subtle hover animations

3. **Pricing Table**
   - 3-tier pricing cards
   - Feature comparison lists
   - Highlighted "popular" tier
   - Annual/monthly toggle

4. **Testimonials**
   - Customer quotes with avatars
   - Company logos
   - Carousel or grid layout

5. **FAQ Accordion**
   - Expandable Q&A items
   - Smooth expand/collapse animations
   - Common questions about your product

6. **Footer**
   - Multi-column navigation links
   - Social media icons
   - Newsletter signup
   - Copyright and legal links

## Tech Stack

- **Tailwind CSS** — Utility-first styling
- **Framer Motion** — Smooth animations
- **React** — Component architecture

## Status

This template is currently in **draft** status. Download functionality will be available in a future update.

When the template becomes available, you'll be able to:
